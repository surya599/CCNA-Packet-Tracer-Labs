# CCNA Packet Tracer Lab 01 – Basic Network Topology

## Overview
This lab focuses on creating a basic network topology in Cisco Packet Tracer. The objective is to become familiar with Packet Tracer's interface, network devices, and automatic cable selection while recreating the topology demonstrated in the CCNA Day 1 training video.

---

## Objective
- Build the given network topology.
- Identify common networking devices.
- Practice placing and connecting devices in Cisco Packet Tracer.
- Use **Automatically Choose Connection Type** for all network connections.

## Notes

- No IP addressing or device configuration was performed in this lab.
- This exercise is intended to familiarize learners with the Packet Tracer workspace and common networking hardware.
- The laptop represents a potential external attacker for future security-related labs.

---

## Learning Outcome

After completing this lab, I can:

- Build a basic enterprise network topology in Cisco Packet Tracer.
- Select appropriate Cisco devices.
- Connect networking devices using Packet Tracer's automatic connection feature.
- Recognize the role of routers, switches, firewalls, servers, PCs, and external hosts within an enterprise network.
