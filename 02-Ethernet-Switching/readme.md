# CCNA Packet Tracer Lab 02 – Network Cabling and Media Selection

## Overview

This lab focuses on selecting the appropriate cable type for connecting various network devices in an enterprise environment. The objective is to understand how different devices communicate and choose the correct transmission media based on the connection type and distance.

---

## Objective

- Build the given network topology.
- Select the correct cable type for every connection.
- Understand when to use copper Ethernet and fiber-optic cables.
- Consider cable distance limitations when choosing network media.

---

## Devices Used

| Device | Quantity |
|---------|---------:|
| Cisco Routers | 4 |
| Cisco Switches | 8 |
| PCs | 3 |
| Server | 1 |

---

## Tasks Performed

- Connected all devices according to the provided topology.
- Selected the appropriate cable for each connection.
- Used fiber-optic cables for long-distance router interconnections.
- Used Ethernet cables for router-to-switch, switch-to-switch, switch-to-PC, and switch-to-server connections where appropriate.
- Considered the specified distances while selecting transmission media.

---

## Key Concepts Learned

- Copper Ethernet cable limitations
- Fiber-optic cable applications
- Single-mode vs. Multimode fiber
- Device-to-device cabling standards
- Enterprise network topology design
- Distance-based media selection

---
