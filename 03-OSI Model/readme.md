# CCNA Packet Tracer Lab 03 – OSI Model and Protocol Analysis

## Overview

This lab focuses on understanding how data travels through a network by analyzing traffic in Cisco Packet Tracer's **Simulation Mode**. The objective is to observe how different protocols operate across the layers of the OSI model and examine the sequence of packets exchanged during normal network communication.

---

## Objective

- Build the given network topology.
- Use Simulation Mode to inspect network traffic.
- Analyze the protocols involved during communication.
- Generate Layer 7 traffic by renewing a PC's IP address.
- Observe packet encapsulation and protocol interactions.
